# Afrolytics 🌍

Open-source African business and economic intelligence dashboard.

![Afrolytics Dashboard](./screenshot.png)

## Features

- 📊 GDP growth analytics
- 💹 Inflation tracking
- ✈️ Tourism insights
- 💰 Startup ecosystem funding
- 🤖 AI-powered summaries
- 🌍 Multi-country selector

## Tech Stack

- React + Vite
- TailwindCSS
- Recharts
- Firebase (coming soon)

## Live Demo

👉 [afrolytics.vercel.app](https://afrolytics.vercel.app)

## Installation

```bash
git clone https://github.com/NoCapBen/Afrolytics.git
cd afrolytics
npm install
npm run dev
```
